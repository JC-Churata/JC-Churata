## About Me

I'm a Systems Engineering student with a strong interest in software development, software architecture, databases, and backend systems. I enjoy building full-stack solutions to real problems and continuously exploring new technologies, from web development to embedded systems and networking.

My work is grounded in solid fundamentals — data modeling, system design, and clean architecture — rather than chasing trends. I'm currently focused on strengthening my backend and full-stack skills while expanding into cloud, DevOps practices, and software architecture patterns.
